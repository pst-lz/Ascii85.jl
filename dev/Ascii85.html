Ascii85.ascii85dec — Method
ascii85dec(in)
converts ASCII85 (Adobe style with <~ ~>) to binary data 
# Arguments
- `in::Array{UInt8}` or `in::String`: the ASCII85 to decode
returns the binary data as Array{UInt8}
